# 모두의창고-IOT
창고의 오프라인에 담긴 정보를 온라인과 연결해주는 IOT파트입니다.
현재 창고 최신화는 창고 정리 때마다 개수를 세면서 그 때 그 때 확인하는 시스템입니다.
하지만 창고 재고와 물품을 온라인에 저장하고, IOT를 바탕으로 불출/입고 때마다 온라인에 정보 처리를 해준다면 보급물자 정리 및 확인해 용이할 것입니다.
