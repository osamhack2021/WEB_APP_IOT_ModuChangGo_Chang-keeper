<앱/웹 구조>

기본 앱 설명
부대 별로 관리자 아이디를 최대 1개 만들 수 있다. 이는 최초 회원가입 시 만들어지며,
최초 회원가입자는 부대 내 관리 창고 내역을 모두 업로드해야 한다.
이후에는 관리자 계정을 인수인계하여 해당 앱/웹을 사용 가능하다.

앱 시작
    로그인 페이지
        or
    회원가입 페이지(최초 1회 실행)
        ->  홈페이지
            1. 창고 현황
                1) 창고 보기
                2) 창고 정보 수정
                3) 창고 추가
                4) 창고 삭제

            2. 입고/출고
                1) 입고
                2) 출고
                3) 입/출고 내역

            3. 마이페이지
                1) 회원정보보기
                2) 회원정보수정

            4. 커뮤니티
                1) 공지사항
                2) QnA

            5. 로그아웃(앱바에 구현)
